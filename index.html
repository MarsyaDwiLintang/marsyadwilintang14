Pengertian

Pengertian Polimorfisme

Polimorfisme adalah prinsip di mana satu antarmuka (nama method/fungsi) dapat digunakan untuk berbagai bentuk tipe data atau objek yang berbeda. Dalam pemrograman OOP, ini memungkinkan kelas turunan mengimplementasikan ulang metode kelas induk

02. Polimorfisme dengan Inheritance

Kita bisa menyimpan objek anak ke dalam tipe parent. Jadi meskipun tipenya sama (misalnya Hewan), hasilnya bisa beda tergantung objeknya.

  @override
  void gambar() => print('Menggambar persegi');

void main() {
  Bentuk bentuk1 = Lingkaran(); 
  Bentuk bentuk2 = Persegi();

  bentuk2.gambar();

10. Studi kasus:Sistem Bentuk Geometrik

sistem bentuk geometrik adalah sistem dalam OOP yang menggunakan konsep abstraksi untuk merepresentasikan berbagai bentuk geometri (seperti persegi, lingkaran, segitiga) dalam satu class dasar, sehingga setiap bentuk memiliki metode yang sama seperti menghitung luas atau keliling tetapi dengan perhitungan yang berbeda.

DART SOURCE CODE
//Nama : Marsya Dwi Lintang
//NIM  : 251410028
//Kelas: SI2A

abstract class Bentuk {
  double hitungLuas();
  double hitungKeliling();

  void info() {
    print('Luas: ${hitungLuas()}, Keliling: ${hitungKeliling()}');
  }
}

class Persegi implements Bentuk {
  double sisi;

  Persegi(this.sisi);

  @override
  double hitungLuas() => sisi * sisi;

  @override
  double hitungKeliling() => 4 * sisi;

  @override
  void info() {
    print('Persegi sisi $sisi, luas ${hitungLuas()}, keliling ${hitungKeliling()}');
  }
}

class Lingkaran implements Bentuk {
  double radius;

  Lingkaran(this.radius);

  @override
  double hitungLuas() => 3.14 * radius * radius;

  @override
  double hitungKeliling() => 2 * 3.14 * radius;

  @override
  void info() {
    print('Lingkaran radius $radius, luas ${hitungLuas()}, keliling ${hitungKeliling()}');
  }
}

void main() {
  Bentuk b1 = Persegi(4);
  Bentuk b2 = Lingkaran(7);

  b1.info();
  b2.info();
}

13. Bonus3

Kode ini menunjukkan penggunaan static, yaitu atribut atau fungsi yang menjadi milik class secara langsung, bukan milik objek. Karena itu, penggunaannya bisa langsung melalui nama class tanpa harus membuat instance terlebih dahulu.

DART SOURCE CODE
//Nama : Marsya Dwi Lintang
//NIM  : 251410028
//Kelas: SI2A

import 'dart:math';

class MathUtils {
  
  // Faktorial
  static int faktorial(int n) {
    if (n <= 1) return 1;
    return n * faktorial(n - 1);
  }

  // Cek bilangan prima
  static bool isPrima(int n) {
    if (n < 2) return false;
    for (int i = 2; i <= sqrt(n).toInt(); i++) {
      if (n % i == 0) return false;
    }
    return true;
  }

  // Pembulatan desimal
  static double roundDouble(double value, int places) {
    double mod = pow(10, places).toDouble();
    return (value * mod).round() / mod;
  }

  // Random integer
  static int randomInt(int min, int max) {
    return min + Random().nextInt(max - min + 1);
  }

  // Random double
  static double randomDouble(double min, double max) {
    return min + Random().nextDouble() * (max - min);
  }

  // Rata-rata
  static double rataRata(List angka) {
    double total = angka.reduce((a, b) => a + b);
    return total / angka.length;
  }

  // Median
  static double median(List angka) {
    angka.sort();
    int n = angka.length;

    if (n % 2 == 1) {
      return angka[n ~/ 2];
    } else {
      return (angka[n ~/ 2 - 1] + angka[n ~/ 2]) / 2;
    }
  }
}

void main() {
  print(MathUtils.faktorial(5));
  print(MathUtils.isPrima(7));
  print(MathUtils.roundDouble(3.14159, 2));
  print(MathUtils.randomInt(1, 10));
  print(MathUtils.randomDouble(1, 5));
  print(MathUtils.rataRata([10, 20, 30]));
  print(MathUtils.median([1, 3, 2, 4]));
}

14. Challenge

Kode ini memperlihatkan kombinasi abstract class dan interface dalam membuat sistem media player yang lebih terstruktur. Abstract class MediaPlayer digunakan sebagai dasar umum, sedangkan Playable dan Pausable berperan sebagai aturan perilaku yang harus diterapkan, sehingga setiap class dapat menjalankan fungsi yang sama dengan cara yang berbeda-beda.Kode ini menunjukkan penggunaan static, yaitu atribut atau fungsi yang menjadi milik class secara langsung, bukan milik objek. Karena itu, penggunaannya bisa langsung melalui nama class tanpa harus membuat instance terlebih dahulu.

DART SOURCE CODE
//Nama : Marsya Dwi Lintang
//NIM  : 251410028
//Kelas: SI2A

abstract class MediaPlayer {
  void play();
}

// Interface Playable
abstract class Playable {
  void start();
}

// Interface Pausable
abstract class Pausable {
  void pause();
}

// AudioPlayer
class AudioPlayer extends MediaPlayer implements Playable, Pausable {
  @override
  void play() {
    print('Memutar audio');
  }

  @override
  void start() {
    print('Audio dimulai');
  }

  @override
  void pause() {
    print('Audio dijeda');
  }
}

// VideoPlayer
class VideoPlayer extends MediaPlayer implements Playable, Pausable {
  @override
  void play() {
    print('Memutar video');
  }

  @override
  void start() {
    print('Video dimulai');
  }

  @override
  void pause() {
    print('Video dijeda');
  }
}

// MAIN (langsung tes semua)
void main() {
  List players = [
    AudioPlayer(),
    VideoPlayer()
  ];

  for (var p in players) {
    p.play();

    if (p is Playable) {
      (p as Playable).start();
    }

    if (p is Pausable) {
      (p as Pausable).pause();
    }

    print('---');
  }
}
